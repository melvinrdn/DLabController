Quickstart
==========

Running the application
-----------------------

Recommended
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
From the repository root:

.. code-block:: powershell

   .\scripts\run.ps1

This activates the virtual environment and launches the application.

Manual launch
^^^^^^^^^^^^^
Once the virtual environment is activated:

.. code-block:: powershell

   dlab

or:

.. code-block:: powershell

   python -m dlab.app
